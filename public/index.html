Introduction

This is a simple real time game of pong controlled with device motion.

Instructions

  1. Navigate device which will serve as screen to 'GAME SCREEN' page from the menu.
  2. Navigate mobile devices with accelerometer or just regular pc with with keyboard to 'CONTROLLER' page.
  3. Now you should see the paddles and game on the main screen.
  4. If you connected with mobile device move it up and down to controll the paddle.
  5. If you are connected with keyboard use arrow keys.
